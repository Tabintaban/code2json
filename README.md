# Конвертер кода в JSON и обратно

Это приложение позволяет конвертировать программный код в JSON-структуру и обратно. Приложение имеет двойное окно: одно для ввода данных, другое для отображения результата.

## Особенности

- Преобразование программного кода в JSON-структуру
- Преобразование JSON-структуры обратно в программный код
- Двухпанельный интерфейс для удобного взаимодействия
- Возможность сохранения и загрузки файлов
- Копирование текста между панелями

## Структура проекта

- `main.py` - основной файл запуска приложения
- `gui.py` - графический интерфейс пользователя
- `logic.py` - логика преобразования кода и JSON

## Использование

1. Запустите приложение: `python main.py`
2. Введите код или JSON в левое окно
3. Нажмите "Преобразовать" для получения результата в правом окне
4. Используйте кнопки для сохранения, загрузки и копирования данных

## Функциональность

- **Преобразование**: автоматическое определение типа ввода (код или JSON) и преобразование в противоположный формат
- **Сохранение**: возможность сохранить результат в файл
- **Загрузка**: загрузка содержимого из файла для обработки
- **Копирование**: копирование текста между левой и правой панелями

## Преимущества вашего проекта:

- Простота использования благодаря графическому интерфейсу
- Двунаправленное преобразование кода и JSON
- Сохранение форматирования и отступов
- Локальное исполнение (безопасно для проприетарного кода)
- Минимальные зависимости (только стандартная библиотека Python)

## Недостатки по сравнению с аналогами:

- Ограниченная функциональность (только базовый анализ строк)
- Поддержка только Python
- Простой синтаксический анализ на основе эвристик
- Отсутствие расширенных возможностей (проверка синтаксиса, подсветка ошибок)
- Морально устаревший интерфейс Tkinter

Проект занимает нишу простого инструмента для базовой конвертации с удобным GUI, подходящего для образовательных целей и локальной обработки кода, но уступает специализированным AST-библиотекам в глубине анализа.
